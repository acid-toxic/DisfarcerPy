# DisfarcerPy

> Ferramenta em Python para gerar perfis fictícios realistas (BR e US), incluindo dados pessoais, documentos, endereço, telefone e user-agent. Permite múltiplos perfis e exportação em JSON.

---

## ⚠️ Aviso

Uso apenas para fins educacionais e testes.  
O usuário é responsável por qualquer uso indevido.

---

```bash
pip install faker
```

```sys```
